# Data Science Salary Explorer

This project is an interactive Dash application that explores salary trends in data science jobs.

## Purpose

The goal of this dashboard is to help users understand how data science salaries change based on company location, experience level, work year, remote work type, job title, and company size.

## Dataset

The dataset used is the Data Science Salaries dataset. The application reads the dataset directly from an online CSV source.

## Features

- Filter by company location
- Filter by experience level
- Filter by work year
- Filter by remote work type
- View top job titles by average salary
- View salary distribution by company size
- View individual salary points by job title
